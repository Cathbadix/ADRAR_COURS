DISPLAY BLOCK

Utilisez la propriété display-block; pour transformer les liens en bloc (pour pouvoir leur donner des dimensions). Donnez à vos liens des dimensions exactes de votre image de fond. Utilisez la propriété line-heiht: px avec une valeur identique à la hauteur du block afin que le texte soit centré verticalement
